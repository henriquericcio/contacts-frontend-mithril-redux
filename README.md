# contacts-frontend-mithril-redux

## Getting start


install deps
```
npm i
```

start frontend app
```
npm start
```

start proxy 
```
docker-compose up
```
*note: proxy working at http://localhost:8080 thats redirects calls to front-end at http://localhost:1234 and to test api container at http://api:80


## tips

### upgrade packages

```
npm outdated
```

```
npm install <package>@latest
```
